# 0.6: New note in Single page app diagram

```mermaid
sequenceDiagram
    participant browser
    participant server

    Note right of browser: User writes a note and clicks Save
    Note right of browser: Browser prevents the default form submit, creates a note, adds it to the page, and sends it to the server as JSON

    browser->>server: POST https://studies.cs.helsinki.fi/exampleapp/new_note_spa with JSON data
    activate server
    Note left of server: Server saves the new note
    server-->>browser: 201 Created
    deactivate server

    Note right of browser: No redirect or full page reload happens
```
